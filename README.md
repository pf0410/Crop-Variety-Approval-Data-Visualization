农作物品种审定数据可视化分析

项目简介
农作物品种审定制度是保障农业生产安全和粮食安全的重要制度。通过对品种审定数据的分析，可以从宏观角度了解我国农作物育种与品种推广的发展趋势。
本项目基于国家农作物品种审定相关数据，使用爬虫获取数据，使用Python进行数据清洗、统计分析与可视化展示，分析不同年份和不同作物类型的品种审定变化情况，并通过图表直观展示数据规律与趋势。


数据来源
数据来源于国家农作物品种审定相关公开数据，主要包含以下信息：
审定年份、作物名称、品种名称、审定信息
在分析前对原始数据进行了整理和结构化处理。

项目目标
本项目主要完成以下分析任务：
分析我国农作物品种审定数量的时间变化趋势
比较不同作物类型的品种审定情况

通过可视化图表展示品种审定结构变化

使用工具
Python（request、Pandas、Matplotlib、Seaborn、JupyterNotebook）


数据分析流程
数据爬虫
从种业大数据平台获取数据

数据整理
导入原始数据
统一字段名称
处理缺失值
转换年份字段格式

数据统计
按年份统计品种审定数量
按作物类型统计品种数量
构建年份×作物的统计表

数据可视化
使用多种可视化方式展示分析结果，包括：
折线图：展示品种审定数量随时间变化的趋势
柱状图：展示不同作物的审定数量
热力图：展示不同年份与作物之间的审定分布情况


主要分析结果
我国农作物品种审定数量整体呈现增长趋势。
水稻、玉米、小麦等主要粮食作物在审定品种中占据较大比例。
不同作物的品种审定在时间上存在一定的集中趋势，而且与地方优势作物正相关，反映出育种重点和农业政策导向。
